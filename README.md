## 首次使用

假设你已经安装好了**对应版本的** StarUML 程序。

1. 第一步：获取 app.asar 文件

   下载[最新发布的 app.asar 文件](https://github.com/SeagullOddy/staruml-custom/releases)。

2. 第二步：替换原版文件

   **小贴士：** 在这之前，推荐你为旧版文件做一下备份哦~

   替换掉原版的 app.asar 文件，其位于 `<程序安装路径>/resources/` 路径下。
